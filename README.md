# Gaelic Coach AI

AI-powered match analysis platform for Gaelic football and hurling coaches.

## Vision

Upload match footage and generate:
- coaching insights
- tactical summaries
- training recommendations
- automated clips
- player development feedback

## MVP Features

- Modern coach dashboard
- Match upload workflow
- AI-generated reports
- Match tagging system
- Video processing pipeline
- Team analytics

## Tech Stack

- Next.js
- TailwindCSS
- TypeScript
- FastAPI (planned)
- OpenAI API
- FFmpeg/OpenCV
- PostgreSQL

## Run Locally

```bash
npm install
npm run dev
```

## Product Direction

The initial focus is workflow acceleration for coaches rather than full autonomous tactical analysis.

Goal:
Reduce post-match analysis time from hours to minutes.
